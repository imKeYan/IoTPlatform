物联网被看做是继PC互联网、移动互联网之后的又一个万亿级的大市场。据市场研究表明，到2025年，连接设备的数量将增加到754.4亿。

物联网云平台作为物联网应用的基础设施，既管理设备、连接用户，又拥有一切数据，必然是各路诸侯的兵家必争之地。

本仓库试图把已知的物联网云平台整理成一个列表，由于物联网平台众多，一个官网一个官网查看审阅，整理不易，**阅读前请你先帮我点个star**⭐支持一下呦。
